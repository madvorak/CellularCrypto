Creates a new totalistic 2D automaton of given ruleset with given initial state.

Parameters
ruleLiveArray that must contain 9 logical values. The value in ruleLive[i] says what happens to a living cell when it has exactly i neighbours alive.
ruleDeadArray that must contain 9 logical values. The value in ruleDead[i] says what happens to a dead cell when it has exactly i neighbours alive.
widthThe width of the new CA (length of rows).
heightThe height of the new CA (number of rows).
rndPseudoRNG instance that will be used to generate the original state.
