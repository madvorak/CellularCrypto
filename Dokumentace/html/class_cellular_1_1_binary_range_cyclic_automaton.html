Detailed Description

Class representing any binary 1D automaton with symmetric scope. The automaton is cyclic - its edges are connected. Therefore, all positions are equivalent.

bool[]  rule,
int  size 
)

Creates a new cyclic CA of a general rule with 000...00100...000 as its initial state.

Parameters
scopeHow many cells on each side from the center determine the next state of the cell. Value 1 makes it equivalent to a ElementaryAutomaton which has rule of size 8. Value 2 means that each new state of any cell depends on five total cells => size of rule must be 32.
ruleArray representing the rule for creating a new state. rule[0] is rule for 0..0, therefore opposite order to the rules of basic automata.
sizeThe size of the new CA.
