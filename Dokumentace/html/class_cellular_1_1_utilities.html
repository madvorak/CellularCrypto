static long Cellular.Utilities.PeriodLengthFast ( CellularAutomaton  CA,
long  limit,
out long  time 
)
static

Calculates a period (number of steps between repetitions) of a CA. This method is fast, but unreliable! It gives only the lower estimate (because of hashing).

Parameters
CAThe examined cellular automaton. Don't worry, this method won't modify it.
limitNumber of steps that will be performed in no repetition is found earlier.
timeOutput parameter telling the time of a first repetition (second occurence of the same state). It is the number of steps done since calling this method (not a real time, not a period length).
Returns
If a period is found within the limit, it returns the length of the period. Otherwise, it returns -1.
